FEATURES IMPLEMENTED
================================================================================

1. UDP PROBING (pkg/probe/udp.go)
   - Sends timestamped UDP packets
   - Measures round-trip time
   - Configurable: target, port, count, interval, payload size, timeout
   - Returns per-probe results with success/failure status
   - Packet format: [seq (4 bytes)][timestamp (8 bytes)][payload]

2. ICMP PROBING (pkg/probe/icmp.go)
   - Raw socket ICMP echo requests
   - Packet ID and sequence number tracking
   - Measures RTT with response verification
   - Requires elevated privileges on Linux

3. JITTER ANALYSIS (pkg/stats/jitter.go)
   - RFC 3550 compliant interarrival jitter
   - Algorithm: J = J + (|D(i-1,i)| - J) / 16
   - Qualitative assessment: Low/Moderate/High
   - Running calculation with O(1) memory

4. LATENCY HISTOGRAM (pkg/stats/histogram.go)
   - Pre-allocated bucket storage
   - Lazy sorting (computed on demand)
   - Percentile computation with linear interpolation
   - Statistics: Min, Max, Mean, StdDev
   - Percentiles: p50, p90, p99, p99.9

5. BUFFERBLOAT DETECTION (pkg/detect/bufferbloat.go)
   - Measures latency under idle conditions
   - Applies background load
   - Measures latency degradation
   - Severity classification:
     * Severe: p99 > 5.0x
     * Moderate: p99 > 3.0x
     * Mild: p99 > 1.5x or p50 > 2.0x
     * None: Below thresholds

6. JSON OUTPUT (pkg/output/json.go)
   - Structured JSON with timestamps
   - Probe results with RTT in milliseconds
   - Statistics embedding
   - Jitter statistics
   - Bufferbloat results

7. TABLE OUTPUT (pkg/output/table.go)
   - Human-readable aligned columns
   - Sections: Results, Statistics, Percentiles, Jitter
   - Formatted with separators
   - Terminal-friendly output

8. CLI INTERFACE (cmd/netprobe/main.go)
   Subcommands:
   - probe:   Send network probes (UDP or ICMP)
     Options: -type, -target, -port, -count, -interval, -payload, -timeout, -output
   - analyze: Detect bufferbloat via load testing
     Options: -target, -idle-count, -load-count, -output
   - listen:  Run UDP echo server
     Options: -port
   
   Global: Help text with examples for all commands

9. UDP ECHO SERVER (cmd/listener/main.go)
   - Listens on configurable UDP port
   - Timestamps received packets
   - Calculates RTT
   - Echoes packets back to sender
   - Displays packet information (sequence, size, RTT)

10. TIMING UTILITIES (internal/timing.go)
    - NowNano(), NowMicro() for current timestamp
    - DurationMicros(), DurationMillis() for conversions
    - HighResTimer struct for elapsed time measurement
    - Microsecond-level precision

ARCHITECTURE HIGHLIGHTS
================================================================================

Clean Go Idioms:
  - Interface-based design where appropriate
  - Error wrapping with context (fmt.Errorf %w)
  - Receiver methods for data operations
  - Proper resource cleanup (defer Close())

Error Handling:
  - Comprehensive error returns on all I/O operations
  - Network errors wrapped with descriptive context
  - Graceful handling of timeout conditions
  - Clear error messages for configuration issues

Memory Efficiency:
  - Pre-allocated slices in histogram (O(n) allocation upfront)
  - Lazy sorting (only when needed for percentiles)
  - O(1) jitter calculation (no history buffer)
  - Reusable buffer allocation in network I/O

Performance:
  - Microsecond-precision timing using time.Now()
  - Minimal allocations in hot paths
  - Efficient percentile interpolation
  - Binary format for network packets (big-endian)

USAGE EXAMPLES
================================================================================

Build:
  go build -o bin/netprobe ./cmd/netprobe
  go build -o bin/netprobe-listener ./cmd/listener

UDP Probing:
  netprobe probe -type udp -target localhost
  netprobe probe -type udp -target 8.8.8.8 -count 30 -output json

ICMP Probing:
  netprobe probe -type icmp -target google.com
  netprobe probe -type icmp -target 8.8.8.8 -count 20

Bufferbloat Detection:
  netprobe analyze -target localhost
  netprobe analyze -target 8.8.8.8 -output json

Echo Server:
  netprobe-listener -port 12345
